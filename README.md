Тут выложены материалы, которые готовились и были представлены на Standoff Talks 17 (https://17.stf365.com/talks/):
/communication - набор промптов для эмуляции общения с топ-менеджментом и формирования бизнес-обоснования проектов по ИБ
/communication_v20_customgpt - набор файлов для реализации Custom GPT (в OpenAI ChatGPT) для создания тренажена общения с топ-менеджментом
/compliance_gpt - набор файлов для реализации Custom GPT (в OpenAI ChatGPT) по 117-му приказу ФСТЭК. Можно перенести и в собственный RAG, например, на базе AnythingLLM
/passowrd_checker - JS-скрипт для проверки надежности пароля и вычисления времени его взлома (запускается целиком локально в браузере - в Интернет пароль не уходит)
/security_policy - пример промптов для генерации и ревью политик для безопасного доступа подрядчиков и безопасного использования ИИ
/threat_modeler_skill - скилл для моделирования угроз, в том числе и по методике ФСТЭК (в процессе доработки)
