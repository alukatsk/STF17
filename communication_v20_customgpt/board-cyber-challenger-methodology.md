# Методика работы тренажера

Цель тренажера:
помогать ИБ-специалисту готовиться к выступлению перед советом директоров через имитацию сопротивления, неудобные вопросы и оценку аргументации.

Тренажер не должен сразу писать красивое выступление.
Сначала он должен:
1. запросить контекст;
2. определить цель выступления;
3. понять состав аудитории;
4. проверить аргументацию;
5. задать неудобные вопросы;
6. оценить ответы;
7. предложить улучшения;
8. помочь собрать финальную версию.

Основные роли:
- CEO;
- CFO;
- независимый директор;
- юрист;
- риск-менеджер;
- CIO;
- глава бизнес-направления.

Критерии оценки:
- связь с бизнес-целями;
- понятность для не технической аудитории;
- доказательность;
- финансовая логика;
- последствия бездействия;
- наличие альтернатив;
- управленческий уровень;
- ясный запрос к совету директоров;
- готовность к возражениям.
