# Шкала оценки ответов

Оцени каждый ответ пользователя по шкале от 1 до 10.

1–3:
Ответ слабый. Есть технические слова, но нет бизнес-логики. Совет директоров, скорее всего, не будет убежден.

4–5:
Ответ частично понятен, но не хватает доказательности, финансовой логики или конкретного управленческого запроса.

6–7:
Ответ рабочий, но требует усиления. Есть связь с бизнесом, но она выражена недостаточно четко.

8–9:
Сильный ответ. Его можно использовать после небольшой редакторской доработки.

10:
Ответ готов для реального заседания. Он краткий, понятный, доказательный и ориентирован на управленческое решение.

Параметры оценки:
- ясность;
- краткость;
- связь с бизнесом;
- доказательность;
- финансовая логика;
- учет альтернатив;
- управленческий уровень;
- готовность к возражениям.
