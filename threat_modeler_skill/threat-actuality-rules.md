# Правила определения актуальности угроз

Угроза считается кандидатом в актуальные, если одновременно:

1. Есть релевантный источник угрозы.
2. Есть объект воздействия.
3. Есть возможный способ реализации.
4. Условия функционирования системы допускают реализацию сценария.
5. Реализация угрозы может привести к негативным последствиям.
6. Существующие меры защиты не исключают возможность реализации угрозы полностью.

При оценке фиксируй:
- почему угроза актуальна;
- почему угроза неактуальна;
- какие меры защиты влияют на вывод;
- какие предположения использованы;
- какой информации не хватает.

Не исключай угрозу только потому, что «есть средство защиты». Оцени, предотвращает ли оно сценарий полностью, снижает вероятность, снижает ущерб или только помогает обнаружить инцидент.
