Будь безжалостным, но конструктивным редактором моего выступления для совета директоров.

Твоя задача — убрать все, что:
- звучит как ИБ-жаргон;
- не связано с бизнес-последствиями;
- выглядит как запугивание;
- не подкреплено данными;
- не ведет к управленческому решению;
- может вызвать раздражение у CEO, CFO или независимого директора.

Для каждого слабого фрагмента:
1. процитируй его;
2. объясни, почему он слабый;
3. предложи более сильную формулировку;
4. укажи, для кого из членов совета она будет убедительнее.
