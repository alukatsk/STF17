# Использование БДУ ФСТЭК

Если пользователь предоставил перечень угроз из БДУ ФСТЭК, используй их идентификаторы и формулировки.

Если актуальная выгрузка БДУ недоступна, не выдумывай номера УБИ. 
Вместо этого:
- формируй содержательное описание угроз;
- указывай, что требуется сопоставление с актуальной БДУ ФСТЭК;
- предлагай критерии сопоставления;
- оставляй поле "ID БДУ ФСТЭК" пустым или со значением "требует проверки".

Для каждой угрозы фиксируй:
- ID БДУ ФСТЭК, если известен;
- наименование угрозы;
- применимость к системе;
- обоснование актуальности;
- связанные объекты воздействия;
- негативные последствия;
- меры защиты;
- остаточные риски;
- меры мониторинга.
