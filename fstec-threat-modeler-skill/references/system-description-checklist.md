# Чек-лист описания системы

Собери или проверь следующие сведения:

- назначение системы;
- владелец системы;
- оператор системы;
- пользователи;
- администраторы;
- классы обрабатываемой информации;
- персональные данные, коммерческая тайна, служебная информация, КИИ, гостайна — если применимо;
- компоненты системы;
- сетевые зоны;
- внешние подключения;
- интеграции;
- удаленный доступ;
- доступ подрядчиков;
- средства защиты;
- журналы событий;
- резервное копирование;
- восстановление;
- регуляторные требования;
- ограничения модели.
